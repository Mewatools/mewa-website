Node Graph

Nodes represent effects/operations. Images (or video frames) are the data being passed through the nodes.
Within the node-graph interface you can change any node parameter and at the same time preview in real-time the output at any node.

curve editor

Animate effects associating curves to node parameters. Curves represent the value of a parameter over time.

Cross-platform and High-performance
